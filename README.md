## 🧭 Logo Usage Guidelines

To maintain consistency and brand integrity, please follow these rules when using the BILD logo:

1. **Do not modify** the logos (color, proportions, or elements).
2. **Keep clear space** around the logos.
3. **Ensure contrast** — don’t place the logos on busy or low-contrast backgrounds.
4. **Do not animate or distort** the logos in any way.

👉 For detailed guidance, refer to the full [BILD Logo Guidelines](https://www.bild.design/foundations/logos).
